# Mechanical Wikipedia

## Overview

Mechanical Wikipedia is a comprehensive web platform designed for Mechanical Engineering students, graduates, educators, researchers, and industry professionals. The platform serves as a centralized knowledge hub that provides engineering subjects, formulas, calculators, reference data, and an AI-powered assistant within a single application.

## Features

### Subject Repository

* Covers major Mechanical Engineering subjects.
* Organized and searchable content for quick access.
* Easy navigation through topic-based categorization.

### Engineering Formula Library

* Large collection of Mechanical Engineering formulas.
* Categorized by subject and engineering domain.
* Mathematical expressions rendered using KaTeX.

### Engineering Calculators

* Interactive calculators for solving engineering problems.
* Instant results with user-friendly input interfaces.
* Supports various mechanical, thermal, fluid, and design calculations.

### AI-Powered Engineering Assistant

* Integrated OpenAI API for intelligent engineering support.
* Provides technical explanations, formula assistance, and concept clarification.
* Uses prompt engineering techniques to generate structured and relevant responses.

### Advanced Search System

* Custom fuzzy-search algorithm for instant content retrieval.
* No external database dependency required.
* Fast and efficient lookup of formulas, subjects, and calculators.

### Voice Search

* Built using the Web Speech API.
* Enables hands-free navigation and content discovery.
* Improves accessibility and user experience.

### Modern User Interface

* Responsive design for desktop, tablet, and mobile devices.
* Built using Glassmorphism-inspired UI components.
* Optimized for readability and ease of use.
